## Introduction
Welcome to the Sidcup Family Golf website project. This project aims to create a visually appealing and responsive website for the Sidcup Family Golf facility. The website provides information about various services offered, including the Toptracer Range, Golf Lessons, Adventure Golf, Coffee Shop, and Leagues.

## Features
- **Responsive Design:** The website is fully responsive and works on various screen sizes, including desktops, tablets, and mobile devices.
- **Interactive Elements:** The website includes interactive elements like hover effects and animations to enhance user experience.
- **Video Background:** A video background is used on the homepage to capture user interest.
- **Scrolling Animation:** Smooth scrolling animations for seamless navigation.
- **Information Sections:** Clear and structured sections providing detailed information about the services offered.

## Technologies Used
- **HTML5:** For the structure of the website.
- **CSS3:** For styling and layout.
- **JavaScript (GSAP):** For animations and interactive elements.
- **Google Fonts:** For typography.
- **Remixicon:** For icons.
